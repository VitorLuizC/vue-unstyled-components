# vue-unstyled-components

## Development

> :exclamation:
> <br>
> You should have **node** and **yarn** available on PATH.

Install dependencies.

```sh
yarn
```

### NPM Scripts

- `build` bundle components down to CommonJS ES5 modules.
  ```sh
  yarn build
  ```
